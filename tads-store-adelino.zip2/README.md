# 🛒 TADS Store

Projeto de e-commerce desenvolvido em React como parte da disciplina Front End 2.

A aplicação simula uma loja virtual com listagem de produtos, busca, filtro, navegação entre páginas e autenticação de usuário.

## 🚀 Como rodar o projeto

```
npm install
npm run dev
```

## 🔐 Login de teste

- Usuário: admin  
- Senha: 123  

## ✅ Funcionalidades

- Componentização com React (Layout, Cabecalho, Rodape, Vitrine, ProdutoCard)
- Consumo de API (DummyJSON)
- Busca de produtos
- Filtro por categoria
- Página de detalhe do produto
- Navegação SPA com React Router
- Página 404
- Sistema de login com Context API
- Rota protegida (/minha-conta)
- Logout e controle de acesso no cabeçalho

## 🎨 Tecnologias utilizadas

- React
- Vite
- React Router DOM
- JavaScript
- CSS

## 📷 Telas do sistema

- Catálogo de produtos
- Detalhe do produto
- Tela de login
- Área protegida

## 📌 Observações

Este projeto utiliza autenticação simulada apenas para fins educacionais.
